Need Help?
The above calendar shows your schedule for the next month. It tells you when to take medication/ go to doctor appointment/ do a test.

1. Reading the calendar.
To read the calendar, click on the date you want to look at. Any events on that date will now be shown. You can switch between months by using the left and right arrows next to the current month and year.

2. Adding reminders
To add a reminder, press the green 'add reminder' button underneath the calendar. Fill in the event name e.g. "Take Diabetes Pills" and select the dates when this will happen. Add the time the event is, using 12 hour time. Press add to add the reminder, or go back to return to the home screen.

3. Deleting reminders
To delete a reminder, press the red 'delete reminder' button underneath the calendar. Scroll using the mouse/trackpad to find the event you want to delete. Press the red cross in the top right hand corner to delete it.
